À propos de la Carte des Événements de Montréal

Cette carte interactive a été conçue pour vous aider à découvrir les événements culturels, sportifs et communautaires qui se déroulent à Montréal. Toutes les informations sont tirées directement du portail des données ouvertes de la Ville de Montréal.

Utilisez les filtres pour affiner votre recherche par date, arrondissement, type d'événement, et plus encore. Cliquez sur un marqueur pour obtenir tous les détails sur un événement, y compris les descriptions, les horaires et les liens pertinents.

Ce projet est open source. Vous pouvez consulter le code source sur GitHub.
